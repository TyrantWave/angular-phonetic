Angular Phonetic 

    
Angular <ng-phonetic phonetic="フォネチック">Phonetic</ng-phonetic>

A simple component to easily render ruby text.

Additionally, if the browser doesn't support ruby text, you can specify the brackets used in the fallback:

Specifying both brackets: 日本語 

  
<ng-phonetic phonetic="にほんご" brackets="「」" >日本語</ng-phonetic>

Using the same bracket twice: 朝鮮的 

  
<ng-phonetic phonetic="조선적" brackets="|" >朝鮮的</ng-phonetic>

Can also strip out the brackets: 中文 

  
  <ng-phonetic phonetic="zhōng wén" brackets="" >中文</ng-phonetic>
